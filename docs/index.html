🍽️ FoodieGram

A recipe sharing app for food lovers

🚀 Features

  • Create & Share Recipes – Add images, ingredients, steps, tags
  • Public Feed – Like and explore recipes from others
  • Search Page – Filter by tags, authors, or title
  • Profile – View your recipes, saved posts, and edit profile
  • Saved Recipes – Bookmark your favorite dishes
  • Private Recipe Notebook – (optional) Save your own drafts privately
  • Authentication – Register and login securely

🧱 Tech Stack

  • Frontend: React, TypeScript, Redux Toolkit
  • Backend: Node.js, Express
  • Database: MongoDB (Mongoose)
  • Other: Axios, Dotenv, Cloudinary (optional)
